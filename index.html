Comment se lancer

Si vous n'êtes pas encore sur⋅e, nous vous invitons à essayer la démo,
et à jeter un oeil aux avantages et inconvénients de gérer un serveur soi-même.

Prenez le temps de réfléchir à quelles applications, quel type de matériel et quel nom de domaine vous souhaitez pour votre serveur.

1. Choisir le support

  • 🪺 Carte ARM / Raspberry Pi
  • 🦕 Vieille tour ou portable (ordinosaure)
  • ☁️ Serveur en ligne (VPS)
Voir le comparatif

2. Installer YunoHost

  • 🚀 Vérifiez les prérequis
    puis flashez ou installez le système
  • 🌐 YunoHost vous guidera pour configurer le premier admin, le nom de domaine et, le cas échéant, votre box internet
